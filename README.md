# Sales Data Analysis (Superstore)

This is a basic data analysis project I made using Python. I used the Superstore dataset from Kaggle to understand sales and profit using simple analysis and graphs.

## What I did

- Loaded the CSV file using pandas  
- Converted the Order Date column into datetime format  
- Removed null values from the dataset  

## Analysis

- Calculated total sales  
- Found top 5 products based on sales  
- Checked sales by category  
- Calculated profit by category  
- Analyzed month-wise sales  

## Visualization

- Created a bar graph to show monthly sales trend using matplotlib  

## Tech Used

- Python  
- Pandas  
- Matplotlib  

## How to run

```bash
pip install pandas matplotlib
python analysis.py
```

## Output

- Identified top-performing products  
- Found category-wise sales and profit  
- Observed monthly sales trends
